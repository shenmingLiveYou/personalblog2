Css选择器 ​

ID选择器 ​

ID是唯一的标识。

在CSS中使用#标识符来定义，在HTML中使用id属性，id的值是标识符

注意： ID具有唯一性，在一个HTML中，一个ID的值能且只能使用一次。

组合选择器 ​

格式：sel1,sel2,sel3...

选择sel1，sel2，sel3让他们拥有相同的代码。

后代选择器 ​

E F，后代选择器，选择匹配的F元素，且匹配的F元素被包含在匹配的E元素内。

__注意： __

  • E和F之间只包含空格。
  • 后代选择器会选中子元素、孙子元素、重孙子元素，只要是后代就行。

链接伪类选择器 ​

链接伪类选择器只针对于HTML中的a标签，它有两种状态：已经访问、未访问过。

  • a:link，匹配a标签，并且a标签具有href属性，并且没有被访问过。
  • a:visited，匹配a标签，并且a标签具有href属性，并且已经被访问过。

注意：

  1. 多个链接指向同一个地址，那么也会受到影响。
  2. 访问过之后就已经存在缓存里了，再打开也是已经访问过。
  3. 可以使用ctrl+shift+del来清理浏览器的记录。

动态伪类选择器 ​

  • E:hover，选择匹配的E元素，且用户鼠标停留在元素上时会被触发。
  • E:active，选择匹配的E元素，且匹配的元素被激活的时候被触发。

注意： 为a连接使用上面4个选择器的时候需要遵循一个爱恨原则LoVe/HAte，link->visted->hover->active

权重值 ​

同一个元素有可能被多个选择器选择，它们又应用了相同的规则（这些规则、属性之间就发生了冲突）。

一个权重值分为4部分：0,0,0,0

  • 行内样式权重值：1,0,0,0

  • ID选择器权重值：0,1,0,0

  • 类、伪类选择器权重值：0,0,1,0

  • 标签选择器权重值：0,0,0,1

  • 通配符选择器权重值：0,0,0,0

  • 后代选择器权重值，就是每个选择器的值相加。

注意：

  • 数值越大，权重越高，权重高的将会生效。
  • 可以忽略权重值，在指定的属性后面使用!important，将不会去计算权重值直接使用这个属性的值。（这个东西能少用就少用）
  • !important->权重值->浏览自带的样式->继承下来的样式

字体和文本属性 ​

字体 ​

所谓的字体就是文字的衣服，外在形式。

我们的字体被安装在C:\Windows\Fonts

字体分为免费使用和商业使用。

  • 字体的安装：

    选中对应字体-》右键-》安装（装完弹框自动就没了）

  • 编辑器字体的设置：

    1. 管理

    2. 设置

    3. 设置字体系列和字体大小

      'Source Code Pro Semibold','Cascadia Code','Fira Code','Anonymous Pro',Consolas, 'Courier New', monospace

    我们通常意义上说的字体有可能不单纯（有可能是字体族，字体族是用来描述粗体、斜体、正常字体等一系列具体风格的字体）

    字体族的类型：

    • 衬线字体：笔画开始、结束的地方有额外的装饰。
    • 非衬线字体：笔画开始、结束的地方粗细差不多。
    • 等宽字体：等宽字体针对于英文。每个字符占据同等的宽度。
    • 手写字体、奇幻字体。

字体系列 ​

格式：font-family:value

注意：

  • 这里设置的字体使用的是客户端的字体，也就是说要保证客户端装了这个字体。
  • 可以给font-family设置一个字体的列表，来定义多个使用的字体，每个字体之间使用逗号进行分隔（从前往后开始找，找到第一个之后就使用了，如果到最后都没有找到该字体将会使用浏览器默认字体）。
  • 设置字体名字的时候我们经常会使用这个字体的英文名字（见表）。

字体大小 ​

没有用css给文本指定尺寸大小，web浏览器会使用它们预先设置好的默认值。大部分浏览器默认值都是16px，我们给他叫基准文本尺寸。

格式：font-size:value

  • px，像素。

    浏览器有时会给网页设置最小的字号一般来说是12px。

    font-size:0可以生效，让文字看不到。

  • em，相对于从父元素继承下来的大小来计算。具体是继承下来的字体大小的多少倍。

  • rem，它的值是基于根元素的文本尺寸来计算。

    在HTML中所谓的根元素就是指的html标签。

字体粗细 ​

格式：font-weight:value

  • normal，正常字体。

  • bold，粗体。

  • 100~900，用它们作为关键字是因为字体有可能映射了9级粗细程度。

    100~900有没有效果要看这个字体具体是否映射了9级的粗细。 很有可能100~400同样粗细，500~900同样粗细。

字体风格 ​

格式：font-style:value

  • normal，正常的字体。
  • italic，显示成斜体。

字体简写格式 ​

格式：font: style weight size family

注意：

  • 简写格式中font-size和font-family是必须要有的
  • font-size和font-family的顺序也不能更改。

文本颜色 ​

格式： color:value

字体下划线 ​

格式：text-decoration:value

  • none，标准的文本，没有下划线。
  • underline，下划线。
  • overline，上划线。
  • line-through，贯穿线。

text-decoration可以使用多个值。text-decoration:underline overline。

字符间距 ​

格式：letter-spacing:value，修改字符之间的距离。

  • normal，相当于letter-spacing:0
  • px，像素。

letter-spacing可以设置为负数。

盒子模型的基本要素 ​

  • width，设置元素的宽度。
  • height，设置元素的高度。
  • background-color，设置元素的背景色。
  • border，设置元素的边框。

元素的显示模式 ​

  • 块级元素（block）

    • 自己独占一行。默认的时候块状元素的宽度填满父元素的宽度。
    • 块状元素可以设置宽度和高度。可以设置的比父元素的宽度、高度大，也可以设置的比父元素的宽度、高度小。
    • 常用的块状元素：div、hn、p、ul、ol、form.....

  • 内联元素、行内元素（inline）

    • 不会自己独占一行，多个行内元素会水平排列，一行放不下就换行。
    • 行内元素是没有办法设置宽度和高度的。
    • 常用的行内元素：span、b、strong、i、em、img....

  • 行内块级元素（inline-block）

    • 内部可以设置宽度和高度，外部和行内元素类似（水平排列）
    • 常用的行内块状元素：表单元素。

  • 改变元素的显示模式

    格式：display:value

    • block，显示为块状元素。
    • inline，设置为行内元素。
    • inline-block，设置为行内块状元素。
    • none，隐藏元素，浏览器不渲染。

文本缩进 ​

格式：text-indent:value

  • px，像素。
  • em，相对于自身的大小来计算。__自身字体大小__的多少倍。

注意：

  1. text-indent的值可以为负数。为负数就不是缩进了，而是突出。
  2. text-indent只会影响元素的第一行，其他行不会有影响。
  3. text-indent可以用在行内块状元素上也可以用在块状元素上，但是不能用在行内元素上。

换行 ​

中文（按字）、英文（按单词）都会换行。white-space决定是否允许换行。

格式：white-space:value用来处理元素内文本是否允许换行。

  • normal，默认值。

    CJK文本遇到容器边界自动换行，非CJK文本由word-break决定。

    china中国、japan日本、korea韩国这三国的缩写CJK。

  • nowrap，不允许换行。

格式：overflow:value，用来设定超出容器之后该如何处理。

  • visible，超出元素框依然可见。
  • hidden，超出之后隐藏。
  • scroll，超出之后的内容以滚动条的方式显示。
  • auto，浏览器自己决定（这个具有不确定性）。

overflow除了刚才所说的这些功能还有其他功能。

格式：word-break（主要针对英文），用来表明怎样进行单词内的断词。

  • normal，浏览器自己来设置是否要进行换行。

  • break-all，让软换行出现在任何字符之间。

    软换行自己换行，硬换行<br/>

水平对齐 ​

格式：text-align:value，控制水平对齐的方式。

  • left，左对齐。
  • right，右对齐。
  • center，水平居中。

在块状元素和行内块状元素上用。

text-align对自身并不会产生影响，会对里面的内容产生影响。

在谁（块状元素、行内块状元素）身上用，对里面的什么元素（文字、行内元素、行内块状元素）生效。

使用场景：

  1. 让其中的文字左、右、居中对齐。
  2. 让其中的行内元素、行内块状元素左、右、居中对齐。

内联盒子：决定了行内元素如何在一行中进行摆放。

内联盒子分为两种：

  • 匿名内联盒子：没有被行内元素标签所包裹，只有光秃秃的文字。
  • 普通内联盒子：已经被行内元素所包裹。

每一个内联盒子分为三部分：内容区、上半行间距、下半行间距。

  • 内容区：用来放置具体的内容，是元素中的字符联合在一起构成的一个方框。

    字体框：浏览器在解释每个字符时，字符都会被包含在一个字体框中，font-size属性控制的就字体框的高度。

    多个字体框连在一起就构成了内联盒子的内容区。

    font-size可以控制内容区大小。

  • 上半行间距、下半行间距：是放在内容区上方和下方的空白，上方和下方的空白是等量的，放在内容区上面的叫上半行间距、下边的叫下半行间距。

    line-height决定了上半行间距和下半行间距。

    (line-height-font-size)/2上、下半行间距的值。

行框：每一行都会有一个行框，这个行框包裹了本行的所有的内联盒子。这个行框的顶边是最高的内联盒子的顶边，底边是最低的内联盒子底边。

<p style="font-size:40px">这里是一个普通的<em style="font-size:50px">em</em>标签！</p>

line-height具有继承性。

line-height放到块状元素上时表示的是行框盒子的最小高度。

格式：line-height：value

  • normal（默认值），浏览器自己决定。font-size的1.2倍左右。

    <span style="font-size:14px;">你好</span>

    (16.8-14)/2

  • px，像素。

    <span style="font-size:14px;line-height:28px">你好</span>

    (28-14)/2

  • 数值，表示的是当前font-size的倍数。

    我们用的时候经常会将line-height用在块状元素上，不加单位只是单纯的数值。设置行之间的距离。

撑开元素高度的是里面的行高，而不是里面的内容。

单行文本垂直居中：已知元素高度的时候将这个元素的行高设置为元素的高度。这种垂直居中方式的弊端就在于只适用于单行文本垂直居中。

1. 动画介绍 ​

1.1 何为动画 ​

​ 动画可以设置变化的次数甚至是无数次，过渡只能看到一次变化过程 如宽度 1000px 变化到 100px，过渡通常需要配合hover使用，而动画不需要，可以自动执行。

​ CSS动画与过渡的相同点就是CSS属性的值在一段时间内发生变化，不同点是动画对变化的方式有更大的控制权（如何重复，重复多少次）。

1. 2 关键帧 ​

​ 众所周知，一段动画，从本质上讲就是一段时间内连续播放的一定数量的画面。每一张画面，我们管它叫做“帧”。一定时间内连续快速播放若干个帧，就成了人眼中所看到的动画。同样时间内，播放的帧数越多，画面看起来越流畅。

关键帧指的是，在构成一段动画的若干帧中，起到决定性作用的2-3帧。

3. 相关属性 ​

  • animation-name 指定关键帧的名字，多个关键字用逗号隔开。

  • animation-duration 指定动画的执行时间，单位是 s。

  • animation-delay 指定对象动画的延迟时间

  • animation-iteration-count 指定动画的具体循环次数

    • number:动画循环次数
    • infinite： 无限循环

  • animation-direction 设置对象动画在循环中是否反向运动

    • normal： 正常方向 (默认)
    • reverse： 反方向运行
    • alternate： 动画先正常运行再反方向运行，并持续交替运行
    • alternate-reverse： 动画先反运行再正方向运行，并持续交替运行

  • animation-fill-mode 设置对象动画时间之外的状态

    • none： 默认值。不设置对象动画之外的状态
    • forwards： 设置对象状态为动画结束时的状态
    • backwards： 设置对象状态为动画开始时的状态
    • both： 设置对象状态为动画结束或开始的状态

  • animation-play-state 设置对象动画的状态

    • running： 运动 (默认)
    • paused： 暂停

  • animation 复合属性，设置多个值

    如果只设置一个时间表示 duration，设置了两个时间分别是 duration 和 delay。其他子属性的值没有数量和顺序要求

响应式布局 ​

响应式布局是什么 ​

​ 伊桑·马科特（Ethan Marcotte）在2010年首先提出了响应式网页设计（RWD,Responsive Web Design）这个术语。在他的一篇文章《Responsive Web Design · An A List Apart Article》中他将已有的三种发开技巧（弹性图片，弹性网格布局，媒体与媒体查询） 进行了整合，命名为响应式网页设计。

所谓响应式页面，是指一套页面，能够适配多种终端，比如宽屏pc电脑，平板，手机，听起来极为的棒，但是，响应式布局极容易出现各式各样的兼容问题，目前社会主流布局方案依然是单独制作移动端页面，所以对响应式，只要抱着能看懂，能做出简单的网页布局即可。

viewport视口 ​

什么是viewport ​

  • viewport 是用户网页的可视区域。
  • viewport 翻译为中文可以叫做"视区"。
  • 手机浏览器是把页面放在一个虚拟的"窗口"（viewport）中，通常这个虚拟的"窗口"（viewport）比屏幕宽，这样就不用把每个网页挤到很小的窗口中（这样会破坏没有针对手机浏览器优化的网页的布局），用户可以通过平移和缩放来看网页的不同部分。

设置viewport ​

前提：

由于历史原因，移动端的浏览器为了让没有做移动适配的网页显示更友好，默认会把浏览器视口宽度缩放为 980px, 不论屏幕宽度是多少。

移动端浏览器的默认缩放行为，会让网页缩小； 如果想要做针对移动端适配的网页就必须不要移动端默认缩放。

设置视口宽度与设备宽度保持一致：

html
<meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1">
<meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1">
1
  • width：控制 viewport 的大小，可以指定的一个值，如果 600，或者特殊的值，如 device-width 为设备的宽度（单位为缩放为 100% 时的 CSS 的像素）。
  • height：和 width 相对应，指定高度。
  • initial-scale：初始缩放比例，也即是当页面第一次 load 的时候缩放比例。
  • maximum-scale：允许用户缩放到的最大比例。
  • minimum-scale：允许用户缩放到的最小比例。
  • user-scalable：用户是否可以手动缩放。

媒体查询运算符 ​

  • and 并且
  • , 或者
  • not 否定 一定要指定媒体类型，因为媒体类型默认all，not后永远返回假
html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        body {
            background-color: #333;
        }
        h1 {
            text-align: center;
            color: #fff;
        }

        /* 视口宽度在 400px 到 500px 之间 */
        @media (min-width: 400px) and (max-width: 500px) {
            body {
                background-color: #f90;
            }
        }
        
        /* 视口宽度小于等于 200px 或 大于等于 1000px */
        @media (max-width: 200px), (min-width: 1000px) {
            body {
                background-color: skyblue;
            }
        }

        /* not  不是屏幕*/
        @media not screen {
            h1 {
                color: red;
            }
        }

        /* only */
        @media only screen {
            h1 {
                color: purple;
            }
        }

        @media only screen and (min-width: 768px) {
            
        }

    </style>
</head>
<body>
    <h1>媒体查询的运算符</h1>
</body>

媒体查询用法 ​

用法一 ​

媒体查询由一种媒体类型组成，并可包含一个或多个表达式，这些表达式可以解析为 true 或 false。

css
@media not|only mediatype and (expressions) {
  CSS-Code;
}
@media not|only mediatype and (expressions) {
  CSS-Code;
}
1
2
3

如果指定的媒体类型与正在显示文档的设备类型匹配，并且媒体查询中的所有表达式均为 true，则查询结果为 true。当媒体查询为 true 时，将应用相应的样式表或样式规则，并遵循正常的级联规则。

方法二 ​

可以针对不同的媒体使用不同的样式表

css
<link rel="stylesheet" media="mediatype and|not|only (expressions)" href="print.css">
<link rel="stylesheet" media="mediatype and|not|only (expressions)" href="print.css">

阈值设置 ​

设置一 ​

两个断点： 640px 1024px

  • 小屏幕 <=640px
  • 中等屏幕 641px ~ 1024px
  • 大屏幕 >1024px

设置二 ​

三个断点： 768px 992px 1200px

  • 手机 超小屏幕 <= 768px
  • 平板 小屏幕 769px ~ 992px
  • 中等屏幕 993px ~ 1200px
  • 大屏幕 > 1200px

设置三 ​

四个断点： 576px 768px 992px 1200px

  • 超小屏幕 <=576px
  • 小屏幕 577px ~ 768px
  • 中等屏幕 769px ~ 992px
  • 大屏幕 993px ~ 1200px
  • 超大屏幕 >1200px

案例练习 ​

html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>响应式设计</title>
    <style>
        /* 
            超小屏幕  container 宽 100%
            小屏幕： 750px
            中等屏幕： 970px;
            大屏幕： 1170px
         */

        * {
            margin: 0;
            padding: 0;
        }
        .container {
            margin: 0 auto;
            height: 200px;
            background-color: pink;

            /* 默认宽度 */
            width: 100%;
        }

        @media (min-width: 768px) {
            .container {
                width: 750px;
                background-color: red;
            }
        }

        @media (min-width: 992px) {
            .container {
                width: 970px;
                background-color: green;
            }
        }

        @media (min-width: 1200px) {
            .container {
                width: 1170px;
                background-color: yellow;
            }
        }

    </style>
</head>
<body>
    <div class="container"></div>
</body>
</html>
