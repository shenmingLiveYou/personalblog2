css预处理器 ​

CSS 预处理器是一个特殊的编程语言，能够编写独有的语法来扩展CSS的语法。

作用：让 CSS 更容易维护,比写起传统的CSS更爽点。

最流行的 CSS 预处理器

  • Sass
  • LESS
  • Stylus

这里以讲解less为例。 less中文文档：https://less.bootcss.com/ 官网： https://lesscss.org/

Less 到底为 CSS 添加了什么功能？

  • 变量声明
  • 嵌套规则
  • 混合(mixin)
  • 父元素选择器&
  • 导入样式
  • ...

安装 Easy LESS 编译器 ​

浏览器默认不识别.less的文件，所以需要借助编译器将less转为原生的css。

可以安装easy less插件进行转换

基本使用：

示例： 项目中创建index.less文件,内容如下：

less
@width: 10px;
@height: @width + 10px;

#box {
  width: @width;
  height: @height;
}
@width: 10px;
@height: @width + 10px;

#box {
  width: @width;
  height: @height;
}

配置文件说明 ​

上面的配置是固定在编辑器中的默认配置，即任何项目都是采用同样的配置。

但是有些项目中目录结构是不一样的，所以上面的写法就固定死了，不够灵活。

我们希望同一个项目在在不同电脑中的vscode编辑器中打开都是采用同样的配置。。

如何配置呢？

我们只需在项目根目录中创建.vscode文件夹,里面创建settings.json文件,这样配置可以跟着项目走。

后面项目在任何电脑打开时，vscode都是同一套配置了。

注意配置文件的优先级：

当前项目配置(.vscode/settings.json) > vscode用户默认配置（settings.json） > 插件默认配置
当前项目配置(.vscode/settings.json) > vscode用户默认配置（settings.json） > 插件默认配置

变量声明 ​

通过变量声明的方式，减少了原来在 CSS 中无法避免的硬编码重复。硬编码相当于代码写死了不够灵活。

示例： 创建index.less文件,内容如下：

less
@width: 10px;
@height: @width + 10px;

#box {
  width: @width;
  height: @height;
}
@width: 10px;
@height: @width + 10px;

#box {
  width: @width;
  height: @height;
}

变量的作用域：

Less 中的作用域与 CSS 中的作用域非常类似。首先在本地查找变量，如果找不到，则从“父”级作用域继承。

less
@var: red;

#box {
  @var: green;
  #list {
    color: @var; // green
  }
}
@var: red;

#box {
  @var: green;
  #list {
    color: @var; // green
  }
}

嵌套 ​

Less 提供了使用嵌套代替传统的层叠能力。 作用： 避免了重复输入父选择器，让复杂的 CSS 结构层次更清晰也便于维护。

假设我们有以下 CSS 代码：

css
#box {   
  color: red; 
} 
#box .header {   
  font-size: 12px; 
} 
#box .footer ul li a {   
  width: 300px; 
}
#box {   
  color: red; 
} 
#box .header {   
  font-size: 12px; 
} 
#box .footer ul li a {   
  width: 300px; 
}

混合（Mixins） ​

混合（Mixin）是一种将一组属性从一个规则集包含（或混入）到另一个规则集的方法。

假设我们定义了一个类（class）如下：

less
.bordered {   
  border-top: dotted 1px black;   
  border-bottom: solid 2px black; 
}
.bordered {   
  border-top: dotted 1px black;   
  border-bottom: solid 2px black; 
}
1
2
3
4

如果我们希望在其它规则集中使用这些属性呢？ 没问题，我们只需像下面这样输入所需属性的类名称即可，如下所示：

less
#menu a {   
  color: #111;   
  .bordered(); 
} 
  
.post a {   
  color: red;   
  .bordered(); 
}
#menu a {   
  color: #111;   
  .bordered(); 
} 
  
.post a {   
  color: red;   
  .bordered(); 
}

导入样式 ​

如某些样式被多个地方同时使用，可以将其定义一个公共的样式文件中，然后进行导入，这样可提高样式的复用性和可维护性。

如下common.less中定义了一些公共样式：

less
// common.less
@bgColor: yellow;
.flex-c-c {
  display: flex;
  justify-content: center;
  align-items: center;
}
.line_ellipsis_1 {
  /*1.先强制一行内显示文本*/
  white-space: nowrap;
  /*2.超出的部分隐藏*/
  overflow: hidden;
  /*3.文字用省略号替代超出的部分*/
  text-overflow: ellipsis;
}
.line_ellipsis_2 {
  overflow: hidden;
  text-overflow: ellipsis;
  /*弹性伸缩盒子模型显示*/
  display: -webkit-box;
  /*限制在一个块元素显示的文本的行数*/
  -webkit-line-clamp: 2;
  /*设置或检索伸缩盒对象的子元素的排列方式*/
  -webkit-box-orient: vertical;
}
// common.less
@bgColor: yellow;
.flex-c-c {

}
