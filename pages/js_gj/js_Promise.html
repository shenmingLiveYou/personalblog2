Promise 承诺 ​

  • Promise 是ES6对异步编程的一种解决方案，比传统的解决方案（回调函数）更合理更强大

  • Promise 可以将异步操作以同步操作的流程表达出来，避免了层层嵌套的回调函数

  • Promise 简单说就是一个容器，里面保存着一个尚未完成且预计在未来完成的异步操作

  • Promise 是一个构造函数，用来创建Promise对象

  • Promise对象代表一个异步操作，有三种状态：

  1. pending（进行中）
  2. fulfilled（成功）
  3. rejected（失败）
  • Promise 对象的状态改变有两种：
  1. 从 pending 变为 fulfilled
  2. 从 pending 变为 rejected

  • 一旦状态改变，就不会再变，任何时候都可以得到这个结果

  • Promise的状态只能改变一次

  • Promise构造函数接受一个函数作为参数(excutor执行器函数)，该函数的两个参数分别是resolve和reject

  • resolve 和 reject 是两个函数，由 JavaScript 引擎提供，不用自己部署

  1. resolve函数的作用是，将Promise对象的状态从“进行中”变为“成功”（即从 pending 变为 fulfilled），在异步操作成功时调用，并将异步操作的结果，作为参数传递出去
  2. reject函数的作用是，将Promise对象的状态从“进行中”变为“失败”（即从 pending 变为 rejected），在异步操作失败时调用，并将异步操作报出的错误，作为参数传递出去
js
var p = new Promise(function(resolve, reject) {// 执行器函数
    // do something ...
    if (/* 异步操作是否成功 */){
        resolve(value) // Promise变为成功状态
    } else {
        reject(error) // Promise变为失败状态
    }
})
p.then(function(value) {
    // 成功时执行
}, function(error) {
    // 失败时执行
})

Promise实例方法 ​

  1. Promise实例的then方法：用于指定成功或失败的回调函数
  • 第一个参数是fulfilled状态的回调函数
  • 第二个参数是rejected状态的回调函数
  • then方法会返回一个新的Promise实例对象（链式调用）
  • 终止链式调用：返回一个进行中状态的Promise对象

  1. Promise实例的catch方法：捕获错误，用于指定发生错误或promise失败状态的回调函数，返回一个新的Promise对象

  2. 不管promise最后的状态如何，在执行完then或catch指定的回调函数以后，都会执行finally方法指定的回调函数

Promise静态方法 ​

  1. Promise.all 可以将多个Promise实例包装成一个新的Promise实例
  • 它接受一个数组作为参数
  • 数组可以是Promise对象，也可以是其它值，但只有是Promise对象时才会等待状态改变
  • 当所有的子Promise都成功，该Promise成功，返回值是全部值的数组
  • 如果有任何一个失败，该Promise失败，返回值是第一个失败的子Promise的结果
js
Promise.all([p1,p2,p3]).then(function (arr){
    console.log(arr)
},function (error){
    console.log(error)
})
Promise.all([p1,p2,p3]).then(function (arr){
    console.log(arr)
},function (error){
    console.log(error)
})

async和await ​

  • async 即 asynchronous 异步的意思

  • 使用 async 修饰的函数叫 异步函数

  • async 函数是什么？一句话，它就是 Generator 函数的语法糖

  • async函数就是将 Generator 函数的星号（*）替换成async，将yield替换成await

  • async和await，比起星号和yield，语义更清楚了

  • async表示函数里有异步操作

  • async函数返回一个 Promise 对象

  • async函数内部return返回的值，是返回的Promise对象的结果

js
async function fn() {
    return 'hello world'
}
fn().then(v => console.log(v)); // "hello world"
console.log('我先执行'); // "我先执行"
console.log( fn() ); // Promise {<resolved>: "hello world"}
async function fn() {
    return 'hello world'
}
fn().then(v => console.log(v)); // "hello world"
console.log('我先执行'); // "我先执行"
console.log( fn() ); // Promise {<resolved>: "hello world"}
