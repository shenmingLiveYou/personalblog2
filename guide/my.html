二、前端学习路线 ​

1、基础知识 ​

基础知识包含学习：HTML、CSS、HTML5、CSS3，建议学习的时间在15天左右，学习完之后建议做以下几个练习，巩固基础知识：

  • 京东图片列表
  • 京东左侧导航条
  • 网易新闻列表
  • 京东轮播图
  • 电影卡片
  • 小米商城首页、详情页、购物车

推荐视频学习地址：https://www.bilibili.com/video/BV1p84y1P7Z5/

2、Bootstrap ​

Bootstrap 让前端开发更快速、简单。所有开发者都能快速上手、所有设备都可以适配、所有项目都适用。Bootstrap 的源码是采用最流行的 CSS 预处理工具 Less 和 Sass 开发的。你的网站和应用能在 Bootstrap 的帮助下通过同一份源码快速、有效地适配手机、平板和 PC 设备，这一切都是 CSS 媒体查询（Media Query）的功劳。建议学习时间2天。

学习完之后建议练习：

推荐视频学习地址：https://www.bilibili.com/video/BV1YW411T7yy

3、JavaScript ​

JavaScript是包含JavaScript基础和JavaScript高级。

JavaScript基础需要掌握ES标准、BOM以及DOM等内容，适合有一定HTML和CSS基础的同学学习。

JavaScript高级主要指掌握原型链、原型、面向对象编程、深拷贝、浅拷贝、事件轮询、闭包、作用域链等知识。

建议学习时间30天-40天。

学习完之后建议练习：

  • 随机图片
  • 表单验证提交
  • 选择放大与缩小
  • 手风琴
  • 九九乘法表
  • 轮播图
  • 选项卡

30个小练习地址：https://github.com/wesbos/JavaScript30.git

推荐视频学习地址(基础)：https://www.bilibili.com/video/BV1YW411T7GX

推荐视频学习地址(高级)：https://www.bilibili.com/video/BV14s411E7qf

4、jQuery ​

jQuery是使用广泛的JS函数库， 它能帮我们程序员简化DOM查询及DOM操作， 轻松实现动画效果。虽然现在不被广泛使用了，但是由于之前很多的项目都是基于jQuery做的，所以还是需要简单的学习一下它的Api。推荐学习时间1天。

推荐视频学习地址：https://www.bilibili.com/video/BV1ts411E7ag

5、ES6+ ​

ECMAScript 6.0（以下简称ES6）是JavaScript语言的下一代标准，已经在2015年6月正式发布了。它的目标，是使得JavaScript语言可以用来编写复杂的大型应用程序，成为企业级开发语言。在工作中使用的很多语法都是属于ES6+的，由于其简便，所以广受开发者的喜爱。这里注意，ES6+并不是需要你学习一门新的语言，ES是ECMAScript的简称，而ECMAScript和JavaScript的关系是，前者是后者的规格，后者是前者的一种实现（另外的ECMAScript方言还有Jscript和ActionScript）。日常场合，这两个词是可以互换的。

推荐视频学习地址：https://www.bilibili.com/video/BV1uK411H7on

注意：Promise也是ES6的新特性，这里需要单独学习，因为其是非常重要的一个模块。

Promise视频学习地址推荐：https://www.bilibili.com/video/BV1GA411x7z1

6、JS模块化 ​

当项目功能越来越多，代码量便也会越来越多，后期的维护难度会增大，此时在JS方面就会考虑使用模块化规范去管理。模块化是为了我们更好的帮助我们开发企业级项目，由于企业级项目它的项目体系很大，所以我们需要考虑到复用性和可阅读性，往往我们会才去模块化的方式进行开发。

推荐视频学习地址：https://www.bilibili.com/video/BV18s411E7Tj

7、NodeJS ​

Node.js是一个能够在服务器端运行JavaScript的开放源代码、是一个跨平台JavaScript运行环境。也就是说NodeJS是通过JavaScript编写，并且能够运行在服务器的语言。

推荐视频学习地址：https://www.bilibili.com/video/BV1bs411E7pD

8、Ajax ​

Ajax 即“Asynchronous Javascript And XML”（异步 JavaScript 和 XML），是指⼀种创建交互式⽹⻚应⽤的⽹⻚开发技术。

Ajax 是⼀种在⽆需重新加载整个⽹⻚的情况下，能够更新部分⽹⻚的技术。

通过在后台与服务器进⾏少量数据交换，Ajax 可以使⽹⻚实现异步更新。这意味着可以在不重新加载整个⽹⻚的情况下，对⽹⻚的某部分进⾏更新。

传统的⽹⻚（不使⽤ Ajax）如果需要更新内容，必须重载整个⽹⻚⻚⾯。

推荐视频学习地址：https://www.bilibili.com/video/BV1WC4y1b78y

9、Axios ​

axios是一个基于Promise的HTTP客户端，可以用在浏览器和node.js中，向服务器发送AJAX请求进行数据交换，是现在前端炙手可热的请求工具。

推荐视频学习地址：https://www.bilibili.com/video/BV1NJ41197u6

10、Git ​

Git 是一个开源的分布式版本控制系统，用以有效、高速的处理从很小到非常大的项目版本管理。

推荐视频学习地址：https://www.bilibili.com/video/BV1NJ41197u6

学习完以上的教程，恭喜你就可以进入框架的学习了，框架是为了我们开发流程更加规范，更加简便，更加哦高效，更加性能的技术，其基础还是基于HTML、CSS、JavaScript这三大基础。因此，学习到这里，笔者希望你已经掌握好了这些技术。

11、React全家桶 ​

React 是 Facebook 开发的一款 JavaScript 库，而 React 被建造是因为 Facebook 认为市场上所有的 JavaScript MVC 框架都不能满足他们的扩展需求， 由于他们非常巨大的代码库和庞大的组织，使得 MVC 很快变得非常复复杂，每当需要添加一项新的功能或特性时，系统的复杂度就成级数增长，致使代码变得脆弱和不可预测，结果导致他们的 MVC 正在土崩瓦解。2011 年 Facebook 软件工程师 Jordan Walke 创建了 React ，并且首次使用 Facebook 的 Newsfeed 功能。做出来以后，发现这套东西很好用，就在 2013 年 5 月开源了。

推荐视频学习地址：https://www.bilibili.com/video/BV1wy4y1D7JT

12、Vue全家桶 ​

Vue是有大神尤雨溪团队进行开发的一款JavaScript库，它是大多数开发者喜爱的一种开源框架，由于其封装较多，所以用起来相对React而言简单，且上手容易。

推荐视频学习地址：https://www.bilibili.com/video/BV1Zy4y1K7SH

最后恭喜你已经具备了可以去企业开发的能力，但是还是需要你多练，多写。并且学习完以上知识能够做一两个企业级的项目。项目非常的多，笔者建议去b站跟着视频做一个自己喜欢的企业级的项目。
